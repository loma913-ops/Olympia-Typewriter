# Olympia Typewriter

![Olympia Typewriter Preview](images/preview.png)

A high-fidelity typewriter font family inspired by the mechanical precision and analog soul of the **Olympia SM3**.

## Overview

Olympia Typewriter is a digital preservation of the "Elite Number 8" typeface commonly found on mid-century Olympia manual typewriters. Designed to bridge between digital convenience and the tactile imperfections of analog machinery, this font family captures the unique character of typebars striking paper.

The project is released under the [SIL Open Font License 1.1](OFL.txt).

## Features

- **Elite Proportions**: Faithfully recreates the spacing and glyph shapes of the Olympia Elite typeface.
- **Analog Character**: Captures the specific geometry and "mechanical soul" of the original Olympia typebars.
- **Digital Ready**: Optimized for modern design workflows, providing a consistent typewriter aesthetic for both print and web.

## Design Philosophy

This project stems from a deep appreciation for the technologies of the past. While analog tools like manual typewriters were once the cutting edge of productivity, today they serve a different purpose: slowing down the mind and focusing on the moment.

Olympia Typewriter seeks to bring that sense of focus and intentionality to the digital workspace, preserving the "Elite" aesthetic for a new generation of writers and designers.

## Installation

The project currently provides fonts in the following formats:

- **TrueType (TTF)**: Located in `fonts/ttf/`
- **Web Fonts (WOFF2)**: Located in `fonts/webfonts/`

Additional formats are part of the ongoing development of the project.
